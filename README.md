# @tscircuit/ti

Bootstrap for the TI component library in the `tscircuit` org.

## Current seed

- `LM358` component wrapper in [lib/LM358.tsx](./lib/LM358.tsx)
- `index.circuit.tsx` seed board that places `LM358` through the TI platform config

## Local usage

```bash
bun install
bun run build
```
